# LB259
Der verwendete Datensatz beschreibt den elektrischen Energieverbrauch eines Stahlindustrieunternehmens über das Jahr 2018. Jede Zeile enthält ein 15Minuten 
Zeitintervall. Der Datensatz enthält unter anderem den Stromverbrauch in Kilowattstunden, Blindleistung, CO2 Emissionen, Leistungsfaktoren sowie 
zeitliche Merkmale wie, Wochentag und Arbeitstagstatus. Ziel des Projekts ist es auf Basis dieser Merkmale den Energieverbrauch (Usage_kWh) vorherzusagen und 
Zusammenhänge zwischen Lasttyp, Zeit und Verbrauch zu analysieren.
